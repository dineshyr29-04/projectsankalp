# Design System - Project Sankalp

## Vision
A minimal, futuristic, and 2D design system with a space-inspired theme. The goal is to feel premium, purpose-driven, and extremely consistent.

## Color Palette
- **Background**: `#05070F` (Deep space)
- **Surface**: `#0B0F1A` (Component backgrounds)
- **Primary**: `#7C5CFF` (Purple accent)
- **Accent**: `#00E0FF` (Cyan highlights)
- **Text Primary**: `#E6E9F2` (High contrast)
- **Text Secondary**: `#9AA4B2` (Low contrast)
- **Border**: `#1A2233` (Subtle definition)

## Typography
- **Primary Font**: Inter (Sans-serif)
- **Headings**: Large, bold, tight tracking.
- **Body**: Minimal text, no long paragraphs, 1.6 line height.

## Spacing
- **Base Unit**: 8px
- **Containers**: Max-width 1280px (7xl)
- **Sections**: Py-20 to Py-32

## Components
- **Navbar**: Floating pill-style, centered, backdrop-blur.
- **Buttons**: Rounded-full, primary gradient or surface background.
- **Cards**: Glass-effect (surface background + border + blur).
